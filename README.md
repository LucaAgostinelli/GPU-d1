# SpMV GPU Investigation - NVIDIA A30 (Ampere)

Experimental study of **Sparse Matrix-Vector Multiplication (SpMV)** on a single NVIDIA A30 GPU.  
Several kernel strategies and sparse formats are compared across 10 SuiteSparse matrices.

## Build

```bash
bash build.sh        # runs cmake + make, output goes to bin/spmv
```

Requirements: CUDA 11.8+, CMake ≥ 3.20, targeting `sm_80` (Ampere).

---

## Run

> **Note:** Before running, update the absolute matrix paths in `ncu.sh`, `sbatch.sh`,
> and `sbatchOne.sh` to match your local environment. Search for the `MATRICES=(...)` 
> array in each file and replace `/home/luca.agostinelli/Deliverable_1/matrices/` 
> with your actual path.

```bash
# Single matrix, all kernels
./bin/spmv <path/to/matrix.mtx>

# Single matrix, one kernel
./bin/spmv <path/to/matrix.mtx> SCALAR        # or S-V-ADAPTIVE, LINE, FLAT, ACC, ELL, HYB, cuSPARSE

# SLURM - one job per matrix (array)
sbatch sbatch.sh

# SLURM - all matrices in one sequential job
sbatch sbatchOne.sh
```

Each run appends one `RESULT_CSV,…` line per kernel to stdout, which `parse_result.py` consumes.

---

## Reproduce CSV results

After running the benchmark jobs:

```bash
# Benchmark results
python3 parse_result.py outputs/          # writes csv/results*.csv

# NCU profiling (requires ncu access)
sbatch ncu_sbatch.sh
bash ncu_extract.sh                       # exports metrics from .ncu-rep files
python3 parse_ncu.py outputs/ncu/         # writes csv/ncu_results*.csv
```

## Kernels at a glance

| Name | Strategy | Format |
|---|---|---|
| `SCALAR` | 1 thread per row | CSR |
| `S-V-ADAPTIVE` | Short rows --> scalar, long rows --> 1 warp per row | CSR |
| `ELL` | 1 thread per row, column-major padded layout | ELL |
| `HYB` | ELL for short rows + COO tail for overflow | ELL + COO |
| `FLAT` | NNZ-splitting, fixed NNZs per block, `atomicAdd` | CSR |
| `LINE` | NNZ-splitting, full rows per block, no atomics | CSR |
| `ACC` | Adaptive dispatcher: selects Flat or Line-enhance | CSR |
| `cuSPARSE` | Vendor reference (`SPMV_ALG_DEFAULT`) | CSR |
